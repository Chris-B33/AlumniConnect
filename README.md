# AlumniConnect – CS4135 Software Architectures

## Project Description
AlumniConnect is a microservices-based platform connecting alumni and students for mentorship, networking, and opportunity sharing.

## Tech Stack
Frontend:
- React (Vite)
- Redux Toolkit
- Axios

Backend:
- Spring Boot
- Spring Security
- JWT
- PostgreSQL / H2
- Spring Cloud Gateway

## Microservices
- User Service
- Mentorship Service
- Event Service
- API Gateway

## Setup Instructions

### Frontend
cd frontend
npm install
npm run dev

### Backend
Run each service individually using IntelliJ or VS Code.

### H2 Console
http://localhost:8080/h2-console
